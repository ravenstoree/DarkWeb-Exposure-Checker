# Email Exposure and Dark Web Security Checker

## Project Overview
This project is a cybersecurity tool developed as part of the Graduation Project requirements. It allows users to verify if their email addresses have been compromised in known data breaches.

## Features
* **Real-time Scanning Simulation:** Mimics the process of checking global leak databases.
* **Cybersecurity UI:** A modern dark-mode interface designed for security professionals.
* **Security Recommendations:** Provides actionable steps for users if a breach is detected.

## Technologies Used
* **Frontend:** HTML5, CSS3 (Custom Styles).
* **Logic:** Vanilla JavaScript (ES6+).
* **Platform:** Hosted via GitHub Pages.

## How to Use
1. Enter a valid email address in the search field.
2. Click on "Scan Now".
3. Review the results and follow the security recommendations provided.
